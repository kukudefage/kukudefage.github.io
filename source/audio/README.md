# 背景音乐配置说明

## 支持的格式
当前系统支持以下标准音频格式：
- MP3 (推荐，兼容性最好)
- WAV
- OGG
- AAC

## 关于KGG格式
**重要提示：** KGG是酷狗音乐的专有加密格式，浏览器**不支持直接播放**。

### 解决方案
如果您有KGG格式的音频文件，需要先转换为标准格式（如MP3）：

1. **在线转换工具：**
   - OnlineAudioConverter (推荐)
   - Zamzar
   - CloudConvert

2. **本地转换工具：**
   - 格式工厂
   - FFmpeg
   - 专门的酷狗格式转换器

## 使用方法
1. 将转换后的音频文件命名为 `background-music.mp3`（或其他支持的格式）
2. 放置在当前目录（`source/audio/`）下
3. 系统会自动加载并播放

## 自定义配置
编辑 `source/js/background-music.js` 文件可以调整：
- 音频文件路径
- 循环播放设置
- 播放器样式
- 自动播放设置
