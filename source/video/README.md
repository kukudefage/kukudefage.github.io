# 背景视频配置说明

## 使用方法
1. 将您的背景视频文件命名为 `background-video.mp4`，并放置在当前目录（`source/video/`）下
2. 支持的视频格式：MP4（推荐，兼容性最好）
3. 建议视频分辨率：1920x1080 或更高，以确保在大屏幕上显示清晰
4. 建议视频大小：小于 10MB，以确保快速加载

## 自定义配置
如果您需要修改视频设置，可以编辑 `source/js/background-video.js` 文件：

- `video.src`: 视频文件路径
- `video.loop`: 是否循环播放（默认：true）
- `video.muted`: 是否静音（默认：true，浏览器要求自动播放视频必须静音）
- `video.playbackRate`: 播放速度（默认：0.8）
- `video.style.opacity`: 视频透明度（默认：0.6）

## 注意事项
1. 浏览器通常要求用户交互后才能播放视频（即使是静音的）
2. 为了最佳性能，建议使用压缩良好的视频文件
3. 考虑为移动设备优化视频大小，或在移动设备上禁用背景视频
